BONDEDCONTEXT:

	- O limite onde um modelo é consistente e uma linguagem ubíqua e valida.
	- "como".
	- linguagem ubíqua.
	- Regras de negócio.
	- Fronteira.

	Exemplo correto:
		- E-commerce. (Domínio)
			- Catálogo	(Subdomínio)
			{
				- Entidade: Pedido
				- Cliente = Quem compra
				- Status =  Criado, Pago, Enviado.
			}
			- Pedido	(Subdomínio)
			{
				- Entidade: Pagamento
				- Cliente = pagador
				- Status =  Autorizado, Capturado, Estornado.
			}
			- Pagamento (Subdomínio)
			- Logística (Subdomínio)
